# 🃏 Мемо

Игра на память для двоих. 15 пар · 30 карточек · Животные.
React + Firebase, деплой на Vercel.

## Деплой (если уже был Зачарованный Лес)

1. Открой `src/firebase.js` — вставь тот же `firebaseConfig` что был у Леса
2. Замени файлы в репозитории на эти
3. Пушни в GitHub — Vercel задеплоит автоматически

```bash
git add .
git commit -m "memo game"
git push
```

## Деплой с нуля

1. **Firebase**: https://console.firebase.google.com → создай проект → Realtime Database → test mode → добавь веб-приложение → скопируй конфиг в `src/firebase.js`
2. **GitHub**: создай репо, залей файлы
3. **Vercel**: https://vercel.com → подключи репо → Deploy

## Как играть

1. Оба открывают ссылку на своих телефонах
2. Один нажимает «Алиса», другой «Алёша»
3. Переворачиваешь 2 карточки — совпали = +1 очко и ещё ход
4. Не совпали — ход переходит сопернику
5. Кто больше пар собрал — победил!

## Локальная разработка

```bash
npm install
npm run dev
```
